# AdobeXD_Smart_Taxi_Mobile_Application

<img  width=500 src="https://user-images.githubusercontent.com/122119114/225603244-693a33cd-8c56-496f-b6bf-35c5399eb817.png">

The Smart Taxi mobile application project aims to design and develop a user-friendly and intuitive application using Adobe XD. The application will allow users to easily book and track their rides in real-time.

The main features of the application will include a user-friendly interface with simple navigation, the ability to select pickup and drop-off locations, track the driver's location, estimate fares, and payment options.
